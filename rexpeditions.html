RexPeditions

This is the first big website that I made. For the showcase in CART we were told to make a website for Jurassic Park. My partner and I worked together for about 2 months. We did research in wireframes, figma and finally programming, with HTML, CSS and Javascript. In this website I learned how to make a cart system, a functional calendar, a timer, a carousel, a mobile layout and more.

I truly enjoyed doing this project and it is one of my best experiences on CSS, I learned a lot while doing this project. Please feel free to explore this website, I include it here not because it is one of my best websites but for all the things I learn with it.
